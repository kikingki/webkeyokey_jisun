# WebKeyOKey(웹키오키)

웹키오스크 서비스를 제공하는 '웹 + 키오스크 = 웹키오키(WEBKEYOKEY)'

편리한 키오스크, 사용하고 싶지만...
비싸고, 가게에 놓을 공간이 없어서 사용할 수 없다구요?
저희 '웹키오키'를 이용하세요!

[웹키오키 최종 github 주소](https://github.com/Skhu-Likelion-8th/WebKeyOKey)
